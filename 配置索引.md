Rules 分为通用规则、中文规则和语言特定规则。

## 快速安装指南

### 核心必装（所有项目）

```bash
./scripts/install-core.sh
```

或手动：

```bash
# Rules
cp rules/common/{coding-style,security,testing,git-workflow,hooks,agents,patterns}.md ~/.claude/rules/common/

# Agents
cp agents/{planner,code-reviewer,tdd-guide,security-reviewer,build-error-resolver}.md ~/.claude/agents/

# Commands
cp commands/{plan,tdd,code-review,build-fix,save-session,resume-session,checkpoint,learn-eval}.md ~/.claude/commands/
```

### 按语言补充

```bash
# Python 项目
cp rules/python/*.md ~/.claude/rules/python/
cp agents/python-reviewer.md ~/.claude/agents/
cp commands/python-review.md ~/.claude/commands/

# Go 项目
cp rules/golang/*.md ~/.claude/rules/golang/
cp agents/{go-reviewer,go-build-resolver}.md ~/.claude/agents/
cp commands/{go-build,go-review,go-test}.md ~/.claude/commands/

# TypeScript 项目
cp rules/typescript/*.md ~/.claude/rules/typescript/
cp agents/typescript-reviewer.md ~/.claude/agents/
```
