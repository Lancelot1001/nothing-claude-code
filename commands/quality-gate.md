# 质量门命令

按需为文件或项目范围运行 ECC 质量管道。

## 使用方法

`/quality-gate [path|.] [--fix] [--strict]`

- 默认目标：当前目录（`.`）
- `--fix`：允许在配置的地方自动格式/修复
- `--strict`：在支持的地方将警告视为失败

## 管道

1. 检测目标的语言/工具。
2. 运行格式化程序检查。
3. 运行 lint/类型检查（当可用时）。
4. 产生简洁的修复列表。

## 注意事项

此命令镜像 hook 行为但是由操作员调用。

## 参数

$ARGUMENTS：
- `[path|.]` 可选目标路径
- `--fix` 可选
- `--strict` 可选
