# 安全规范

## 提交前必须检查

- [ ] 无硬编码密钥（API key、密码、Token）
- [ ] 所有用户输入已验证
- [ ] SQL 注入防护（参数化查询）
- [ ] XSS 防护（HTML 净化）
- [ ] CSRF 保护已启用
- [ ] 认证/授权已验证
- [ ] 所有端点有速率限制
- [ ] 错误消息不泄露敏感数据

## 秘密管理

- **绝不**硬编码秘密
- 使用环境变量或秘密管理器
- 启动时验证必需的秘密
- 暴露后立即轮换

## 安全响应协议

发现安全问题时：
1. **立即停止**
2. 使用 **security-reviewer** agent
3. 修复严重问题后再继续
4. 轮换任何暴露的秘密
5. 检查代码库中是否有类似问题
