# 企业agent运维

对于需要超越单个 CLI 会话的操作控制的云托管或持续运行的agent系统，使用此skill。

## 运营领域

1. 运行时生命周期（启动、暂停、停止、重启）
2. 可观测性（日志、指标、追踪）
3. 安全控制（范围、权限、终止开关）
4. 变更管理（发布、回滚、审计）

## 基线控制

- 不可变部署产物
- 最小权限凭证
- 环境级密钥注入
- 硬超时和重试预算
- 高风险操作的审计日志

## 需要追踪的指标

- 成功率
- 每个任务的平均重试次数
- 恢复时间
- 每个成功任务的成本
- 故障类别分布

## 事件模式

当故障激增时：
1. 冻结新发布
2. 捕获代表性追踪
3. 隔离故障路由
4. 用最小安全更改进行修补
5. 运行回归和安全检查
6. 逐步恢复

## 部署集成

此skill与以下内容配合：
- PM2 workflow process
- systemd 服务
- 容器编排器
- CI/CD 门控
