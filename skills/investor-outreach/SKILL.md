# 投资者外联

编写简短、个性化且易于操作的投资者沟通。

## 何时激活

- 给投资者写冷邮件
- 起草温暖介绍请求
- 在会议或无回应后发送跟进
- 在流程中撰写投资者更新
- 根据基金 thesis 或合作伙伴契合度定制外联

## 核心规则

1. 个性化每条出站消息。
2. 保持请求低摩擦。
3. 使用证明，而非形容词。
4. 保持简洁。
5. 绝不发送可发给任何投资者的通用副本。

## 冷邮件结构

1. 主题行：简短且具体
2. 开场：为什么是这个投资者
3. 推介：公司做什么、为什么是现在、什么证明重要
4. 请求：一个具体的下一步
5. 结尾：姓名、职位，如需要一个可信度锚点

## 个性化来源

参考以下一项或多项：
- 相关投资组合公司
- 公开的 thesis、演讲、文章或帖子
- 共同联系
- 与投资者重点的明确市场或产品契合

如果缺少该上下文，询问或说明草稿是等待个性化的模板。

## 跟进节奏

默认：
- 第 0 天：初始出站
- 第 4-5 天：带一个新数据点的简短跟进
- 第 10-12 天：带干净结束的最终跟进

除非用户想要更长的序列，否则在那之后不要继续催促。

## 温暖介绍请求

让连接者更轻松：
- 解释为什么介绍是合适的
- 包含可转发的简介
- 保持可转发简介在 100 字以内

## 会议后更新

包含：
- 讨论的特定内容
- 承诺的答案或更新
- 如有可用的一个新证明点
- 下一步

## 质量门控

交付前：
- 消息是个性化的
- 请求是明确的
- 没有空洞或乞求语言
- 证明点是具体的
- 字数保持精简
